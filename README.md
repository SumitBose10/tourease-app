# Tour Ease 🌍✨

Tour Ease is a modern, smart tourism and transportation platform. It helps travelers seamlessly discover popular destinations, optimize their travel routes based on selected attractions, and book the perfect vehicle for their journey.

## 🚀 Features

- **Dynamic Route Optimization**: Select a destination and your preferred attractions, and our system dynamically generates the most efficient routes (e.g., Heritage Trails, Budget Adventures) with estimated costs and times.
- **Vehicle Marketplace**: Compare and choose from a wide range of vehicles including Cars, Bikes, Scooters, Shuttles, and Ride Shares. Sort by price, rating, or ETA.
- **Seamless Booking Flow**: A beautiful and intuitive booking experience that carries your destination and route preferences through to checkout.
- **Premium User Interface**: Built with modern aesthetics in mind, featuring dark mode, glassmorphism, gradient accents, and smooth animations powered by Framer Motion.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 📂 Project Structure

- `/src/app/page.tsx` - The stunning landing page.
- `/src/app/explore/page.tsx` - Browse and discover destinations.
- `/src/app/plan/page.tsx` - Itinerary planning and route optimization.
- `/src/app/vehicles/page.tsx` - Vehicle marketplace and comparison.
- `/src/app/booking/page.tsx` - Checkout and trip confirmation.

## 🏃‍♂️ Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 💡 How it Works

1. **Explore**: Start by browsing curated destinations like Goa, Manali, or Jaipur.
2. **Plan**: Select specific attractions you want to visit. The system generates optimized routes tailored to your selections.
3. **Select Transport**: Choose a route and pick the best vehicle for your needs from the marketplace.
4. **Book**: Complete your booking securely with your trip details automatically pre-filled.
